# Support Policy

## Scope

This repository provides the official PayTabs PHP SDK for Payment Gateway integrations.

## Support Channels

- Integration/Product support: https://site.paytabs.com/en/contact-us/
- Security vulnerabilities: https://ai.paytabs.com/en/paytabs-bug-bounty/

Do not post security vulnerabilities in public GitHub issues.

## Supported Release Window

- Latest stable release: fully supported
- Previous minor release line: best-effort support for critical fixes

## Response Expectations

Support response times can vary by request type and channel.
For production-impacting incidents, use official PayTabs support channels with clear impact details.
